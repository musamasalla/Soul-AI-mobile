Upload Instructions

1 Log in to your Supabase Dashboard

Go to https://app.supabase.com and log in to your account.

2 Select your project

Select the "Soul AI" project from your dashboard.

3 Navigate to Storage

In the left sidebar, click on "Storage" to access the storage management interface.

4 Select or Create the "podcasts" Bucket

If the "podcasts" bucket already exists, click on it. If not, create a new bucket named "podcasts":

  1. Click the "New Bucket" button
  2. Enter "podcasts" as the bucket name
  3. Make sure "Public" is selected for the bucket type
  4. Click "Create bucket"

5 Upload the Placeholder Audio File

Once you're in the "podcasts" bucket:

  1. Click the "Upload" button
  2. Select the file from your computer at:
    /Users/musamasalla/Documents/Soul AI/placeholder-audio/generating-placeholder.mp3
  3. Make sure to rename the file to generating-placeholder.mp3 if it's not already named that
  4. Click "Upload"

6 Verify the File is Accessible

After uploading:

  1. Click on the uploaded file in the bucket
  2. In the file details panel, you should see a "URL" field
  3. Copy this URL and verify it's accessible by pasting it in a new browser tab

The URL should look something like:
